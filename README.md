# foro-3-
El wireframe es una herramienta muy importante en el desarrollo de aplicaciones móviles porque permite visualizar la estructura de las pantallas antes de iniciar la programación. Gracias al wireframe, los desarrolladores pueden organizar mejor la interfaz, identificar posibles errores y mejorar la experiencia del usuario.
Además, el wireframe ayuda a planificar la navegación de la aplicación y facilita la comunicación entre los integrantes del proyecto. También permite ahorrar tiempo durante el desarrollo, ya que muchos cambios pueden corregirse en la etapa de diseño y no cuando la aplicación ya está programada.

Explicación del diseño

La aplicación BeerStock Colombia tendrá una interfaz sencilla y moderna para facilitar el uso por parte de los usuarios. El diseño está pensado para que los bares y pequeños negocios puedan administrar su inventario de forma rápida y eficiente.
Las principales pantallas de la aplicación serán:

•	Login 
•	Menú principal 
•	Inventario 
•	Agregar producto 
•	Reportes 

Dentro de la aplicación, el usuario podrá:

•	Ver inventario 
•	Registrar productos 
•	Actualizar stock 
•	Buscar cervezas 
•	Consultar precios 

Además, el administrador podrá:

•	Agregar productos 
•	Eliminar productos 
•	Modificar cantidades 
•	Revisar inventario 
•	Ver reportes básicos 

La aplicación incluirá diferentes marcas de cerveza como:

•	Águila 
•	Poker 
•	Club Colombia 
•	Costeña 
•	Pilsen 
•	Andina 
•	Corona 
•	Heineken 
•	Budweiser 
•	BBC
